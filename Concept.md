**# Mengled**



**Mengled is a 2D Skyrim like open world game set in a medieval fantasy world.**

**It is built by using 4 self written modules:**



**- core handling startup, versioning and window settings**

**- engine providing structs and functions to handly gameobjects aswell as resource management**

**- editor for generating levels and prefabs**

**- game handling game logic**

**- utils providing basic utilities like file handling, logging etc.**



**## Game idea**



**### Movement**



**- basic zelda like wasd movement**

**- special keys for dashing and teleporting**



**### Combat**



**- Melee**

**- Ranged**

**- Magic**

**- Weapon switching using mouse wheel**



**### Inventory**



**- Minecraft like toolbar**

**- Sections for clothing, weapons, consumables, materials**

**- Filter options**

**- Magic tab**



**### Crafting**



**- Smithing**

**- Tailoring**

**- Leatherworker**

**- Alchemy**



**## Graphics**



**- Simple 2D style using grid based textures**

**- Texture size of 16x16 pixels as base**

**- Simple effects and not overloaded (keep visibility)**

**- Colorful**



**## Gameplay**

**- Fast paced combat**

**- No time based -> make gameplay outside of combat slow**

**- Huge enemy masses to fight**

**- Technical bosses instead of bullet sponges**
